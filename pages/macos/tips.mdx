# Tips

## Add dock separators

Add one separator.

```shell copy
defaults write com.apple.dock persistent-apps -array-add '{ "tile-type" = "spacer-tile"; }'
```

Reload Dock.

```shell copy
killall Dock
```

Reset Dock.

```shell copy
defaults delete com.apple.dock; killall Dock
```

Reload Finder.

```shell copy
killAll Finder
```
