# Browser

Browsers, extenstions, and other stuff related to browsing the web.

## Browsers

- **[Arc](https://arc.net/)** • Key features: workspaces, profiles, vertical tabs, split screen, AI features and more. Based on Chromium.
- [Brave](https://brave.com/)
- [Firefox](https://www.mozilla.org/en-US/firefox/new/)
- [Google Chrome](https://www.google.com/chrome/)
- [Microsoft Edge](https://www.microsoft.com/en-us/edge)
- [Opera](https://www.opera.com/)
- [Safari](https://www.apple.com/safari/)
- [Tor Browser](https://www.torproject.org/download/)
- [Vivaldi](https://vivaldi.com/)
- **[DuckDuckGo](https://duckduckgo.com/app)** • Browser from DuckDuckGo search.
- **[Sidekick](https://www.meetsidekick.com/)** • A productivity browser for focused work.
- **[SigmaOS](https://sigmaos.com/)** • Key features: workspaces, profiles, vertical tabs, tabs as tasks, split screen, ad-free, magic theme, AI features and more. Based on WebKit.

## Extensions

- **[uBlock Origin](https://github.com/gorhill/uBlock)** • Blocks ads, trackers, coin miners, popups, annoying anti-blockers, malware sites, etc.
- **[Unhook](https://unhook.app/)** • Remove YouTube Recommended, Shorts, suggestion feeds, comments, and more.
