# Terminal

## Terminal Emulators

- **[Alacritty](https://alacritty.org/)** • A terminal emulator for BSD, Linux, macOS and Windows.
- **[iTerm2](https://iterm2.com/)** • A terminal emulator for macOS.
- **[Rio](https://raphamorim.io/rio/)** • Fast, 24-bit true color, Images in Terminal, Cross-platform, Font ligatures.
- **[Tabby](https://tabby.sh/)** • An infinitely customizable cross-platform terminal app for local shells, serial, SSH and Telnet connections.
- **[Terminal.app](https://support.apple.com/guide/terminal/welcome/mac)** • The terminal emulator included with macOS.
- **[Warp](https://www.warp.dev/)** • A terminal reimagined with AI and collaborative tools for better productivity. • [Docs](https://docs.warp.dev/) • [Terminus](https://www.warp.dev/terminus)
- **[Wave](https://www.waveterm.dev/)** • An Open-Source, AI integration. Render anything inline. Save sessions and history. Powered by open web standards.

## Prompt

- **[Oh My Posh](https://ohmyposh.dev/)** • A prompt theme engine for any shell.
- **[Powerlevel10k](https://github.com/romkatv/powerlevel10k)** • A prompt theme for Zsh.
- **[Spaceship](https://spaceship-prompt.sh/)** • Minimalistic, powerful and extremely customizable Zsh prompt.
- **[Starship](https://starship.rs/)** • The minimal, blazing-fast, and infinitely customizable prompt for any shell.

## Nerd Fonts

**[Website](https://www.nerdfonts.com/)**

### Installation by Homebrew

Run the following command to see available Nerd Fonts:

```sh
brew search nerd-font
```

Install a Nerd Font with Homebrew:

```sh
brew install --cask <font-name>
```

Example:

```sh
brew install --cask font-jetbrains-mono-nerd-font
```
