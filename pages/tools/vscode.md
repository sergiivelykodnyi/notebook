## Keyboard Shortcuts

| Command                   | Key           |
| ------------------------- | ------------- |
| Keyboard shortcuts editor | `cmd+k cmd+s` |

### Fold

| Command             | Key           |
| ------------------- | ------------- |
| Toggle fold         | `cmd+k cmd+l` |
| Fold                | `alt+cmd+[`   |
| Unfold              | `alt+cmd+]`   |
| Fold all            | `cmd+k cmd+0` |
| Fold level 1 (1..7) | `cmd+k cmd+1` |

### Markdown

| Command                   | Key       |
| ------------------------- | --------- |
| Show preview side by side | `cmd+k v` |
